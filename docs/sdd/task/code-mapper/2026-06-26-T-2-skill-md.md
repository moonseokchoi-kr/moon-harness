## 의존 태스크
없음 (Wave 1, T-1과 병렬 실행 가능)

**주의**: 3-상태 분류 기준 표는 T-1의 `classify_probe_state()` 구현과 동일 기준이어야 한다. 두 태스크가 병렬 실행되므로 아래 기준을 이 태스크에서 SSOT로 정의하고, T-1 엔지니어는 SKILL.md를 참조하여 구현한다:
- healthy: 인덱스 통계(노드/엣지 수, ready 등) 반환
- not_initialized: "not initialized"(또는 동등 미초기화 신호) 포함
- unavailable: 도구 미등록/연결 오류/미지 신호 (fail-safe)
